## 1️⃣0️⃣❎ Edge Output Stream extensions

This repo contains the various extension classes provided by the 10x engine [Edge flavor](https://doc.log10x.com/architecture/flavors/#edge).

This repo includes support for the following [modules](https://doc.log10x.com/architecture/module/):

- Event [appender outputs](https://doc.log10x.com/api/output/#log4j2).
- Time-series [Micrometer outputs](https://doc.log10x.com/api/output/#micrometer).
