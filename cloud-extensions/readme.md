## 1️⃣0️⃣❎ Cloud IO Streams and Compile extensions

This repo contains the various extension classes provided by the 10x engine [Cloud flavor](https://doc.log10x.com/architecture/flavors/#cloud).

This repo includes support for the following [modules](https://doc.log10x.com/architecture/module/):

- Log analyzer [Apache Camel inputs](https://doc.log10x.com/api/input/#apache-camel).
- Object Storage [access](https://doc.log10x.com/api/input/#object-storage) as well as java.io backing stream classes for the [index output](https://doc.log10x.com/run/input/objectStorage/index/) and [query input](https://doc.log10x.com/run/input/objectStorage/query/) modules.
- Time-series [Micrometer outputs](https://doc.log10x.com/api/output/#micrometer).
- Compile [executable inputs](https://doc.log10x.com/api/compile/#executable).
