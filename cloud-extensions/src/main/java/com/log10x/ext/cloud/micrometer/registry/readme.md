## 1️⃣0️⃣❎ Time-series output extensions

This folder contains the registry factory classes for [time-series](https://doc.log10x.com/run/output/metric/#extensions) output modules bundled with the 10x engine [cloud](https://doc.log10x.com/architecture/flavors/#cloud) flavor.

To learn more see [Micrometer Output API](https://doc.log10x.com/api/output/#micrometer).
